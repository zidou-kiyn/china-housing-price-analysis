# 数据源反爬调研（Playwright 实测，2026-07-06）

## 结论速览

| 站点 | 可访问性 | 反爬强度 | 数据粒度 | 加载方式 | 适用定位 |
|------|---------|---------|---------|---------|---------|
| 链家 lianjia.com | ❌ 直接 `Forbidden` | 极强（IP/请求安全检查） | 小区/房源级明细 | SSR | 仅作演示级小样本目标 |
| 安居客 anjuke.com | ❌ 跳 58 antibot 验证码 | 极强（58 antibot + 滑块） | 小区/房源级明细 | SSR + 加密 | 仅作演示级小样本目标 |
| 中国房价行情 creprice.cn | ✅ 正常访问 | 弱 | 城市/区县/街镇级 + 历史月度时序 | **SSR HTML table 直出** | **主力真实数据源** |

## 实测细节

### 1. 链家（lianjia.com）
- `https://sh.lianjia.com/ershoufang/` 和 `https://www.lianjia.com/` 均被重定向到
  `https://hip.lianjia.com/forbidden?...reason=您的请求未能通过我们的安全检查`。
- 无 Cookie / 指纹的裸请求直接封禁。需要完整浏览器指纹 + 登录态 + 代理池才可能突破，成本高、法律与稳定性风险大。

### 2. 安居客（anjuke.com / 58 系）
- `https://quanzhou.anjuke.com/community/` 立即 302 到
  `https://callback.58.com/antibot/verifycode?...`（"请输入验证码"）。
- 58 antibot 体系，触发即需人工滑块，脚本化困难。

### 3. 中国房价行情（creprice.cn）——推荐主力源
- 首页宣称"超过 20 年动态更新的全国城市房产市场数据""2005 年开始"。
- URL 规则清晰、可推导：
  - 城市选择页：`/rank/citySel.html`
  - 城市/区县房价行情：`/urban/{cityPinyinAbbr}.html`（如泉州 `qz`）
  - 小区列表：`/web/halist/{city}.html`
  - 城市房价排行：`/rankprice/sale/{id}.html`
  - 房价分布图：`/market/pricefenbu/flag/{id}.html`
- **关键优势：历史时序数据以 SSR HTML `<table>` 直出**，`requests + lxml/BeautifulSoup` 即可解析，无需破解 API、无需 Selenium。示例（泉州市辖区页 `/urban/qz.html` 实测抓到）：

  | 时间(年-月) | 供给(元/㎡) | 关注(元/㎡) | 价值(元/㎡) |
  |---|---|---|---|
  | 2026-06 | 14,286 | -- | 8,596 |
  | 2026-05 | 14,194 | 18,267 | 8,515 |
  | 2026-04 | 14,578 | 12,547 | 8,643 |
  | 2025-12 | 15,588 | 18,200 | 8,824 |
  | ... | ... | ... | ... |

  另有价格区间分布表（如 14000-15000 占 13.77%）。页面用 ECharts 渲染，但**原始数据在 table 里已可直接取**。
- 小区级 `/web/halist/qz.html` 明细可能需登录/付费（实测列表未渲染出价格），细粒度小区数据从中介站补充或用其它公开渠道。

## 对系统设计的影响（支撑"混合数据源策略"）

1. **真实爬取主力 = creprice.cn**：静态 SSR，`requests+lxml`，负责城市/区县级 **历史月度均价时序**（走势分析、ML 训练的核心输入）。反爬弱、稳定、可长期增量。
2. **中介站（链家/安居客）= 演示/教学目标**：用 Playwright/Selenium 做**小规模、低频**抓取，展示"多源采集 + 反爬处理"能力；不作为数据主力，避免因反爬导致链路不稳。
3. **公开数据集 = ML 建模底座**：Kaggle House Prices 数据集验证回归范式；必要时补充国家统计局 70 城房价指数等公开时序。
4. 采集层需抽象成**统一 Source 接口 + 可插拔适配器**，使"静态源 / 动态源 / 数据集导入"三类走同一套清洗-入库-特征管线。
