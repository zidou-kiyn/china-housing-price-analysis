### 路由守卫

```typescript
router.beforeEach((to, from, next) => {
  const auth = useAuthStore()
  if (to.meta.requiresAuth && !auth.isLoggedIn) {
    next({ name: 'login', query: { redirect: to.fullPath } })
  } else if (to.meta.requiresAdmin && auth.role !== 'admin') {
    next({ name: 'home' })
  } else {
    next()
  }
})
```

## 4. 图表规格

### 4.1 均价柱状图（PriceBarChart）

| 属性 | 说明 |
|------|------|
| 场景 | 区县列表均价展示 |
| X 轴 | 区县/板块名称 |
| Y 轴 | 均价（元/㎡） |
| 颜色 | 供给价（蓝）/ 价值价（绿） |
| 交互 | tooltip 显示具体数值、同比变化 |

### 4.2 走势折线图（TrendLineChart）

| 属性 | 说明 |
|------|------|
| 场景 | 单区域历史走势 |
| X 轴 | 年月（YYYY-MM） |
| Y 轴 | 均价（元/㎡） |
| 线条 | 最多三条（供给/关注/价值），可勾选 |
| 交互 | tooltip、dataZoom 时间范围滑块 |

### 4.3 多区域对比折线图（CompareLineChart）

| 属性 | 说明 |
|------|------|
| 场景 | 2~5 区域走势叠加 |
| X 轴 | 年月 |
| Y 轴 | 均价 |
| 线条 | 每区域一条，颜色区分，图例标注 |
| 交互 | tooltip、legend 切换、dataZoom |

### 4.4 价格分布图（DistributionChart）

| 属性 | 说明 |
|------|------|
| 场景 | 区域内价格区间占比 |
| 模式 | 饼图（默认）/ 柱状图（可切换） |
| 数据 | `[{range: "5000-8000", percentage: 25.3}]` |
| 交互 | tooltip 显示百分比与数量 |

### 4.5 预测走势图（PredictChart）

| 属性 | 说明 |
|------|------|
| 场景 | 历史+预测走势 |
| X 轴 | 年月 |
| Y 轴 | 均价 |
| 历史段 | 实线 |
| 预测段 | 虚线 + 置信区间阴影（areaStyle） |
| 分界 | 今/历史分界标记线（markLine） |

### 4.6 地图热力图（HeatMap）

| 属性 | 说明 |
|------|------|
| 场景 | 城市内各区县均价分布 |
| 底图 | ECharts geo，离线 GeoJSON（福建省/泉州市） |
| 着色 | visualMap 连续色阶（低→绿，高→红） |
| 交互 | 点击区域 → 下钻到区县详情页 |

## 5. GeoJSON 方案

- **来源**：DataV.GeoAtlas 或 geo.datav.aliyun.com 下载各城市区县级 GeoJSON。
- **存放**：`frontend/public/geo/` 目录，按城市 code 命名（如 `qz.json`）。
- **加载**：首次进入地图页时异步加载并注册到 ECharts：
  ```typescript
  const geoJson = await fetch(`/geo/${cityCode}.json`).then(r => r.json())
  echarts.registerMap(cityCode, geoJson)
  ```
- **离线优先**：GeoJSON 随前端部署，不依赖外部 CDN。

## 6. 状态管理（Pinia）

### 6.1 auth store

```typescript
interface AuthState {
  token: string | null
  user: { id: number; username: string; role: string } | null
}
```

- `login(username, password)` → 调用 API → 存 token 到 localStorage → 更新 state
- `logout()` → 清除 token 与 state
- `isLoggedIn` / `role` getter

### 6.2 city store

```typescript
interface CityState {
  cities: City[]
  currentCity: City | null
  districts: District[]
}
```

- `fetchCities()` → GET /cities
- `fetchDistricts(code)` → GET /cities/{code}/districts

### 6.3 price store

```typescript
interface PriceState {
  currentPrice: PriceInfo | null
  trendData: TrendPoint[]

  rankList: RankItem[]
  compareData: CompareRegion[]
}
```

### 6.4 predict store

```typescript
interface PredictState {

```

## 7. API 请求封装

```typescript
// api/index.ts
const api = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || '/api/v1',
  timeout: 10000,
})

api.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

api.interceptors.response.use(
  response => response.data,
  error => {
    if (error.response?.status === 401) {
      const auth = useAuthStore()
      auth.logout()
      router.push({ name: 'login' })
    }
    return Promise.reject(error)
  }
)
```

## 8. 响应式布局

- **断点**：Element Plus 默认断点（xs < 768px, sm, md, lg, xl）。
- **布局**：`el-row` + `el-col` 栅格；图表容器使用 `resize-observer` 自适应。
- **大屏**：DashboardView 使用 CSS Grid，图表按比例缩放。
