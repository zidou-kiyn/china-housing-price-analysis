# 多源采集 —— 研究综合与实现决策

> 汇总 4 份研究（govstats / kaggle-datasets / agency-antibot / integration-design）的跨源结论与最终实现决策。

## 决定性的全局约束：缺"中国大陆出口 IP"

四份研究独立指向**同一个根因**——本机（OVH 欧洲机房 IP）与所给代理（美国数据中心 IP）都是**境外 + 机房 IP**：

| 源 | 本环境可达性 | 根因 |
|---|---|---|
| **Kaggle 公开数据集** | ✅ **可直接下载**（无需登录/token，302→签名 GCS，已实测） | 全球可达 |
| **creprice**（现源） | ⚠️ 可达但**间歇 IP 限流**（SSL EOF/502） | 机房 IP 高频被限速，非硬封 |
| **国家统计局** easyquery | ❌ 403 WAF 地理围栏（`reason:UrlACL`） | **任何 Header 组合都无法让境外 IP 拿数据**，必须大陆 IP |
| **链家/贝壳** `/xiaoqu/` | ❌ pg2 起 302，~15 次后整机 IP 封禁 | 缺住宅 IP（链家甚至不给验证码，直接 Forbidden） |
| **安居客** | ❌ 首请求即 58antibot + 易盾滑块 | 缺住宅 IP + 滑块 |

**所给代理（`10.0.0.1:2260` → 美国 208.87.x/134.195.x/103.62.x）对政府站和中介站全部 000/RST，比直连更差**，不是解决方案。

**结论**：短期内本环境唯一能"新增并产出真实数据"的可靠源是 **Kaggle 公开数据集**。政府源/中介站的采集代码可以写好，但**live 数据需要用户提供中国大陆出口 IP（住宅或云主机）**才能跑通。

## 实现决策（据此重排子任务）

### ✅ child A — 多源框架泛化 + 数据源切换（已完成）
地基。解耦硬编码源名、能力声明、按能力编排、切换 API + 前端卡片、`price_snapshot.source` 注记列。creprice 零回归。

### ✅ child C — Kaggle/公开数据集导入（**提为首要新源**）
本环境可验证、可产出真实数据。首选 `ruiqurm/lianjia`（北京 2011–2017，31.8 万行成交，可聚合真实月度 ¥/㎡ 序列）+ `vnvile/shanghai-second-hand-house-dataset`（上海 2023-10 快照，MIT 许可，含挂牌/评估双价）。
- 能力：`{CITIES, DISTRICTS, PRICE_TIMELINE}`（京沪有区县）；`price_unit=cny_per_sqm`。
- 定位：**历史回填 + 交叉校验**，非实时替代（数据截止 2017/2023）。
- 许可：北京 CC BY-NC-SA（非商业），上海 MIT；仅用于本系统分析展示，需在 UI/文档标注来源与许可。

### ⚠️ child B — 国家统计局数据源（**实现适配器 + 离线测试，live 门控于大陆 IP**）
政府源最可靠、无验证码，但境外 IP 403。实现 `GovStatsSource`（easyquery 封装 + 解析 + 用录制 JSON 的离线单测），注册进框架并在切换 UI 可见（`price_unit=index`），但**live 抓取需大陆 IP**。数据是 70 城指数（非 ¥/㎡、无区县），MVP 存入独立轻表或仅作占位，不污染 creprice 的 ¥/㎡ 列。
- 交付价值：拿到大陆 IP 即可用；解析逻辑经离线测试保障。
- 明确不做：指数与 ¥/㎡ 跨源混算、指数的前端专门展示（后续任务）。

### ⏸️ child D — 中介站直采（**暂缓，记录理由**）
链家/贝壳/安居客数据形状最贴合，但本环境无法开箱采集：缺国内住宅 IP、无可用开源库（`jumper2014/lianjia-beike-spider` 已删库，fork 全停留 2018-19 失效）、安居客还需破易盾滑块。ROI 最低、最易碎、高维护。**暂缓**，待用户提供稳定国内住宅代理池后再评估最小可行版（贝壳 `/xiaoqu/` 聚合 + Playwright 真机 + 严格限速）。

## 给用户的建议（非本次可自动解决）
1. **要跑通政府源 + 稳定 creprice**：提供一个**中国大陆出口 IP**（住宅代理或国内云主机中转），填入管理端「采集代理」。当前美国代理无效。
2. Kaggle 历史数据已足以立即丰富京沪深的历史深度并交叉校验 creprice。
3. 中介站直采建议最后再做，且必须先有住宅代理池。
